This is the first video game that I ever created that I made using Unity, It is a simple platformer style game that I made in my beginning GIMM courses. we were tasked to make a game based on a similar genre of 2D game we have played before and create three simple physics changes or object movement changes to make the game have action. It was my first experience messing with game coding and making a game and for a first project it turned out alright and started my progress in becoming a game designer.
